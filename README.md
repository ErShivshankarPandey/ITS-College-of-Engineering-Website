# ITS-College-of-Engineering-Website
"Responsive multi-page static website for ITS College of Engineering, built with plain HTML, CSS &amp; JS. Includes a home page, course catalogue, per-course registration forms, student/admin login, about &amp; contact pages, plus a testimonial wall, image slider, and accessibility widget."

├── index.html              # Home page
├── about.html               # About the college
├── courses.html              # Full course catalogue
├── select-course.html        # Course picker → routes to registration forms
├── contact.html               # Contact form + campus details
├── login.html                  # Student login
├── admin-login.html            # Admin login
├── ba-register.html            # BA registration form
├── bca-register.html           # BCA registration form
├── bcom-register.html          # B.Com registration form
├── bsc-register.html           # B.Sc registration form
├── btech-register.html         # B.Tech registration form
├── mba-register.html           # MBA registration form
├── mca-register.html           # MCA registration form
├── index.css                  # Shared stylesheet for all pages
├── index.js                    # Shared behaviour (widgets, forms, slider, feedback)
